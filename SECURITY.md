# Security Policy

This repository primarily contains documentation and examples.

## Reporting a vulnerability

If you believe you have found a security vulnerability in this repository (for example, in example scripts or referenced automation), please:

- Avoid opening a public issue with sensitive details.
- Open a minimal issue without exploit details, and request a private follow-up.

Maintainers will respond as soon as reasonably possible.

## Supported versions

Only the latest contract major version is considered actively maintained.
